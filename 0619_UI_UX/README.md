무선 인터넷에 대한 이통3사의 기득권이 사라지고, 수평식/상향식 프로세스로 바뀌기 시작했다.

순식간에 스마트폰으로 대부분의 업무를 할 수 있게 바꾸었다.

   - ux를 만들고 무언가 기획한다는 것의 시작은 다른 사람에 대해 이해하는 것이다.
